# Furniture store

Интернет магазин по продаже мебели на Django

## Features

- Фильтр для каталога товаров
- Динамическое обновление корзины покупок 
- Личный кабинет
- Админ панель

## Technologies

- Python
- Django
- PostgreSQL
- JavaScript
- CSS
- HTML
